# Security Policy

ScriptScore is a public preview. Please report vulnerabilities privately before public disclosure.

## Reporting

Open a private security advisory on GitHub when available, or contact the maintainers through the security contact listed on the project profile.

Include:

- Affected component and version or commit.
- Reproduction steps.
- Impact assessment.
- Whether any student data, credentials, or local databases may be affected.

Do not include real student data, API keys, Canvas credentials, or raw exam material in reports. Use synthetic examples.
